# Шаблон мини-аппа

Стартовая заготовка для нового приложения. Всё базовое (тема, кнопки, карточки,
анимации, хаптика) уже в общем дизайн-ките `apps/assets/`.

## Как сделать новый апп

1. Скопируй папку `_TEMPLATE` → переименуй, напр. `apps/timer`.
2. В `index.html` поменяй:
   - `<title>` и шапку (`h1`, `.sub`);
   - блоки контента под свою задачу (лишнее удали);
   - ссылку и текст в блоке `.ref` (или убери, если рефка не нужна).
3. Логику пиши в нижнем `<script>`. Доступны хелперы:
   - `App.haptic('light'|'medium'|'success'|'error'|'select')` — вибро;
   - `App.toast('текст')` — всплывашка снизу;
   - `App.openModal('id')` / `App.closeModal('id')` — модалка;
   - `$('#id')`, `$$('.class')` — выбор элементов.
4. Добавь карточку аппа в каталог `apps/index.html`.
5. Деплой: из папки `apps/` → `git add . && git commit -m "..." && git push`.
   Через ~1 мин живёт на `…github.io/mini-apps/<папка>/`.

## Дизайн-кит (классы из `assets/app.css`)

- Текст: `h1`, `h2`, `.sub`, `.label`, `.muted`, `.small`
- Карточка: `.card`
- Кнопки: `.btn` + `.btn-primary` / `.btn-ghost` / `.btn-card`
- Список: `.list` → `.row` (`.k` / `.v`)
- Каталожные плитки: `.tile`
- Чипы: `.chips` → `.chip`, кнопки-пресеты `.chip-btn`
- Ввод: `.field`
- Бейджи/точки: `.badge-ok|warn|bad`, `.dot ok|warn|bad`
- Рефка: `.ref`
- Модалка: `.overlay#id` → `.modal`
- Загрузка: `.spin`, `.skeleton`

Меняешь цвет/радиус в `assets/app.css` → меняется во всех аппах сразу.
